# Artifact availability

The project artifact contains:

- versioned English--Indonesian query sets and a revision audit;
- retrieval, calibration, gap-analysis, robustness, and review-evaluation results;
- provisional LLM relevance judgments, annotation guidelines, and decision notes;
- scripts required to rebuild indexes, reproduce tables, and compile the paper;
- hashes and environment metadata embedded in versioned result files.

Large pretrained model weights and original source PDFs and videos are not
redistributed. They are downloaded or accessed from their named upstream
providers. Derived transcript excerpts and figure crops are included for audit;
copyright remains with the original publishers and creators. The build scripts
and metadata identify the sources needed to reconstruct the collection.

The public project repository is:

https://github.com/shaban2/Crosslingual_Scholar_Search

The permanent Zenodo concept DOI, which resolves to the latest archived
release, is:

https://doi.org/10.5281/zenodo.21427042

The archived `v1.0.1` release has the version-specific DOI
https://doi.org/10.5281/zenodo.21427043. The concept DOI should be used in the
paper and arXiv metadata so that the citation remains valid across later
releases.

See `REPRODUCIBILITY.md` for commands and `ETHICS_AND_DATA.md` for data-use notes.
